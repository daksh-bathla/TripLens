# TripLens Frontend (v2.0)

This is the revamped TripLens frontend, built with React, Vite, and Tailwind CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

## Structure
- `src/components`: Reusable UI elements.
- `src/pages`: Main application screens (Dashboard, NewTrip, TripDetails, etc).
- `src/App.tsx`: Main routing and layout.
- `src/index.css`: Global styles and design system tokens.
